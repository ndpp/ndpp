User's Guide
============

Welcome to the NDPP User's Guide!  This tutorial will guide you through the
essential aspects of using NDPP to convert continuous-energy data in to a 
form which can be readily tallied in a Monte Carlo neutron transport code with a 
significantly enhanced Figure-of-Merit.
